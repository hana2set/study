# [Spring Cloud](https://spring.io/projects/spring-cloud)란?

분산 시스템에서 사용되는 패턴을 빠르게 구축하도록 돕는 도구들. 
- 버저닝
- 서비스 레지스트레이션, 디스커버리
- 라우팅
- 로드밸런싱
- MSA

> [!note]
> Nexflix에서 MSA를 위한 Spring Cloud Netflix에 구성요소들을 다양하게 지원해왔으나 현재 대부분이 중단된 상태.(Kubernetes 등의 영향으로 생각됨)  
> 2025년 6월 기준 Eureka 서버 외엔 존재하지 않음으로, 다른 서비스로 마이그레이션하는 것 추천.
