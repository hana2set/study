# [n8n](https://n8n.io/)
노드 기반 워크플로우 자동화 플랫폼.
<img width="1066" height="603" alt="Image" src="https://github.com/user-attachments/assets/2db5b7da-87ba-4715-a97d-5e6bdcec0535" />

## 다른 워크플로우 자동화 툴(Zapier, Make(Integromat)) 과의 차별점
1. 유연성과 확장성
   - 오픈소스 기반
   - 커스텀 코드 작성(무한한 자유도)
   - 자체 노드 개발(개발에 참여 가능)
2. 강력한 데이터 처리 능력
3. 비용 효율성 및 데이터 주권
   - 셀프 호스팅
   - 데이터 프라이버시


| 구분 | n8n | Zapier | Make (Integromat) |
| :--- | :--- | :--- | :--- |
| **핵심 특징** | 오픈소스, 셀프호스팅, 코드 기반 확장 | 간편함, 가장 많은 앱 연동 | 시각적 시나리오 빌더, 복잡한 흐름 |
| **가격 정책** | 커뮤니티 버전 무료 (셀프호스팅) | 실행 횟수 기반 유료 구독 | Operation 수 기반 유료 구독 |
| **데이터 처리** | 다중 아이템, 복잡한 분기/병합에 강함 | 단일 아이템 처리에 최적화 | 유연하지만 n8n보다 다소 복잡 |
| **확장성** | JS 코드로 커스텀 로직, 노드 개발 가능 | 제한적 (Code by Zapier) | 제한적 (Custom Apps) |
| **주요 타겟** | 개발자, 엔지니어, 데이터 분석가 | 마케터, 비개발 직군 | 숙련된 기획자, 자동화 전문가 |
| **데이터 주권** | **완벽 보장 (자체 서버 운영)** | 외부 서버에 데이터 전송 | 외부 서버에 데이터 전송 |
