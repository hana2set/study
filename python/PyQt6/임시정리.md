- pyqt6 vs. pyside6
https://www.pythonguis.com/faq/pyqt6-vs-pyside6/

1. 관대한 PGPL 라이선스로 제공하기 위해 PySide 생김
2. PySide가 개발이 튀쳐지게 됨
3. Qt프로젝트에서 PySide를 공식으로 채택
4. Qt6에서 두가지가 합쳐짐 (바인딩 가능해짐)
5. 소프트웨어 배표 계획에 따라 골라쓰면 됨
   1. 상업용 목적 -> PyQt6 구매 OR PySide6 사용
   2. 개인 개발 -> PyQt6 사용해도 문제 X (GPL 라이선스)
